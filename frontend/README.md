# A6CARS - Frontend (Vite + React + Tailwind)

## Setup
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open browser at the address shown (usually http://localhost:5173)

Notes:
- Tailwind is configured in tailwind.config.js.
- API base is `http://localhost:4000/api` (change in src/api.js if backend runs elsewhere).
